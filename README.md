# AI Code Explainer Bot 
- Project Overview
AI Code Explainer Bot is an AI-powered application developed using Python and Gemini API. The system helps users understand programming code by generating simple explanations and line-by-line analysis automatically using Large Language Models (LLMs).
The application accepts source code from the user and provides AI-generated explanations using Google Gemini AI.
# Module 1 – User Interface
- The screen where the user interacts
- Input box, buttons, and output display
# Module 2 – Code Explanation
- Explains what the code does in simple words
-  Acts like a teacher explaining step by step
# Module 3 – Debugging
-  Finds errors in the code
-  Suggests fixes and explains the mistakes**
